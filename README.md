# 📊 Contoso SQL Sales & Profitability Analysis

## 📌 Objective
Analyze sales performance, profitability, and business efficiency using the Contoso dataset through SQL queries.

## 🛠️ Tools & Technologies
- SQL (T-SQL)
- SQL Server
- Data Modeling (Fact & Dimension tables)

## 📊 Analysis Performed
- Sales analysis by store and country
- Profitability analysis (Revenue, Cost, Profit, Margin)
- Top-performing stores
- Revenue breakdown by continent

## 💡 Key Insights
- High-revenue stores do not always have the highest margins
- Profitability varies significantly across countries
- Some locations show high volume but low efficiency

## 📁 Project Structure
- SQL queries organized by analysis type
- Focus on business-driven insights
